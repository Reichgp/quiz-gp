# quiz-gp 🧠

Interactive self-assessment test engine for **GP (Gestión de Proyectos / Project Management)**.

This repository contains an original self-assessment platform and curated question set created by the repository owner with AI-assisted content generation, validation and refinement workflows.

Part of the DevQuiz-Hub ecosystem.

## 📚 Core Topics Covered

* **Block 1**: Project fundamentals, project lifecycle, stakeholders and project success criteria.
* **Block 2**: Project planning, scope management, scheduling, budgeting and quality management.
* **Block 3**: Project leadership, communication, negotiation, conflict resolution and team management.
* **Block 4**: Risk management, uncertainty, governance and project monitoring.
* **Block 5**: PMBOK principles, project methodologies, programs, portfolios and organizational value delivery.

## 🛠️ Data Contract & Logic

This repository implements the standardized architecture of the organization:

* **`gp_questions.json`**: Holds the structured question schema, metadata and explanatory content.
* **`quiz.js`**: Core asynchronous engine handling in-place shuffling, dynamic data fetching and client-side state management.
* **`index.html`**: Zero-dependency SPA frontend built with responsive modern CSS variables.
